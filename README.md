# Penzion Žižka - statický web s poptávkou rezervace

Web je původní statická šablona projektu zachovaná ve stejné skladbě sekcí. Obsah, logo, fotky, ceny a kontakty jsou přizpůsobené pro Penzion Žižka v Chrudimi.

Použité soubory:

- `index.html`
- `styles.css`
- `script.js`
- `privacy-policy.html`
- `vylety/`
- fotky a logo v `assets`

Formulář je poptávkový. Odeslání formuláře neznamená automaticky potvrzenou rezervaci.
